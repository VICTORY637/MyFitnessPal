Структура на изхода на програмата:

ГЛАВНО МЕНЮ:

	1. Log in
	2. Sign in
	3. Quit


--------------------------------------------------------

If (1)

	Въвежда ИМЕ и ПАРОЛА.
	
	Cout << Welcome, user!
	Cout << Standard или Premium
	
	Cout << Случаен мотивиращ цитат на деня.
	
	Cout << Цел за деня - брой приети калории от целевите
	
	МЕНЮ.2:
		a. Прогрес за деня
			Cout << Текущ баланс - приети
			Cout << Препоръчителен прием
			Cout << Списък с храненията, тренировките и водата за текущия ден
			i. Редактирай прогрес
			ii. Назад -> МЕНЮ.2
		b. Редактирай прогрес
			A) Добави хранене
				1) Храна
					a) Добави само калори
					b) Детайлно добавяне
			B) Изтрий хранене
			Cout << Кое хранене искаш да изтриеш
			Cout << Списък с хранения за деня
					a) Въвеждаш цифра на съответното
			C) Добави тренировка
					a) Тренировка
					b) Калории
			D) Изтрий тренировка
				Cout << Коя тренировка искаш да изтриеш
				Cout << Списък с тренировките за деня
					a) Въвеждаш цифра на съответната
			E) Добави вода
					a) Колко чаши
			F) Изтрий
					a) Колко чаши
			G) Назад -> МЕНЮ.2
		c. История
			i. Виж прогрес за специфична дата
				1) If (има данни за тази дата)
					Cout << Текущ баланс - приети
					Cout << Препоръчителен прием
					Cout << Списък с храненията, тренировките и водата за текущия ден
					a) Назад -> МЕНЮ.2
				2) If (няма)
					Cout << Не са намерени данни
					a) Потърси друга дата
					b) Назад -> МЕНЮ.2
			ii. Изтрий прогреса за специфична дата
			iii. Назад -> МЕНЮ.2
		d. Моят акаунт
			Cout << Потребителски данни, физически характеристики, въведени в началото
			i. Промени физическите характеристики
			ii. Смени User information
			iii. Назад -> МЕНЮ.2
		e. Logout -> връща в МЕНЮ.1
